# The Motion Club

Premium animated portfolio landing page for **The Motion Club** — a lifestyle movement community built around running, yoga, trekking, cycling, and wellness.

## Stack

- React 19 + TypeScript
- Vite 5
- Tailwind CSS 3
- Framer Motion
- GSAP + ScrollTrigger
- Lenis smooth scroll
- Lucide React

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Scripts

| Command         | Description              |
| --------------- | ------------------------ |
| `npm run dev`   | Start development server |
| `npm run build` | Production build         |
| `npm run preview` | Preview production build |

## Sections

1. **Hero** — Cinematic video, animated headline, mouse glow, particles
2. **Marquee** — Scroll-driven dual-row activity gallery
3. **About** — Story, animated text reveal, live counters
4. **Vision & Mission** — GSAP scroll-reveal split cards
5. **Activities** — Six activity cards with hover effects
6. **Why TMC** — Interactive feature grid
7. **Gallery** — Masonry layout with hover overlays
8. **Our Journey** — Sticky scroll timeline
9. **Join CTA** — Glowing call-to-action
10. **Footer** — Links and social
